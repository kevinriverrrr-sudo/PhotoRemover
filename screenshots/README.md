# Screenshots

This directory contains screenshots of the PhotoRemover application.

## How to Add Screenshots

1. Take screenshots of the application
2. Name them descriptively:
   - `main.png` - Main interface with 3D background
   - `upload.png` - File upload area
   - `processing.png` - Image being processed
   - `result.png` - Final result with download button
   - `mobile.png` - Mobile responsive view

3. Add them to this directory
4. Update the README.md to reference them

## Required Screenshots

- [ ] Main interface (desktop)
- [ ] Upload interface
- [ ] Processing state
- [ ] Result with transparent background
- [ ] Mobile responsive view

## Guidelines

- Use high-quality PNG format
- Resolution: At least 1920x1080 for desktop
- Show the 3D animated background
- Include realistic example images
- Blur any sensitive information (API keys, etc.)
